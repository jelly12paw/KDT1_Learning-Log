### CASE문
> 특정 상황에서 데이터를 변환하여 활용
>
> `ELSE`를 생략하는 경우 NULL값이 지정
>
> 파이썬의 if문과 비슷

```sql
CASE
    WHEN 조건식 THEN 식
    WHEN 조건식 THEN 식
    ELSE 식
END
```

## 서브쿼리
> 특정한 값을 메인 쿼리에 반환하여 활용
>
> **단일행 서브쿼리**, **다중행 서브쿼리**, **다중컬럼 서브쿼리**가 있다.

```sql
SELECT *
FROM 테이블
WHERE 컬럼1 = (
    SELECT 컬럼1
    FROM 테이블
    WHERE ~
);
```

### 1. 단일행 서브쿼리
> 서브쿼리의 결과가 0 또는 1
>
> 단일행 비교 연산자와 함께 사용 : 
>`=`, `<`, `<=`, `>=`, `>`, `<>`

### 2. 다중행 서브쿼리
> 서브쿼리 결과가 2개 이상
>
> 다중행 비교 연산자와 함께 사용 :
> `IN`, `EXISTS`, `ALL`, `ANY`, `SOME`
